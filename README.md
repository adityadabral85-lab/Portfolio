# Personal Portfolio Project

A simple personal portfolio website made with HTML, CSS, and JavaScript.

## Features

- Responsive navigation menu
- Profile image section
- About section
- Skills section
- Project cards
- Contact information
- Contact form interaction
- Resume download button placeholder

## Files

- `index.html` - Website structure and page content
- `style.css` - Styling, layout, colors, and responsive design
- `script.js` - Mobile menu and form behavior

## How to Run

Open `index.html` in your browser.

If you use VS Code, you can also run it with the Live Server extension.

## Customize

Update these details in `index.html`:

- Your name
- Your profile image
- Your skills
- Your project names and links
- Your email and phone number
- Your resume file link

If you want the resume button to work, add your resume PDF to this folder and name it `resume.pdf`.

## How to Push to GitHub

```bash
git init
git add .
git commit -m "Add personal portfolio project"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git
git push -u origin main
```

Replace `YOUR-USERNAME` and `YOUR-REPOSITORY` with your GitHub account and repository name.
